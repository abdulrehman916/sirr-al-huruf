## 📝 MALAYALAM EXPLANATION STRUCTURE

Each Malayalam explanation includes:

### 1. **പേര് (Title)**
The name of the rule/planet/mansion in Malayalam

### 2. **അർത്ഥം (Meaning)**
What this rule means in simple terms

### 3. **ഉപയോഗം (Use)**
How to use this rule in practice

### 4. **ഗുണങ്ങൾ (Benefits)**
List of benefits from following this rule

### 5. **ചെയ്യേണ്ട കാര്യങ്ങൾ (Suitable Actions)**
What actions are recommended during this time

### 6. **ഒഴിവാക്കേണ്ട കാര്യങ്ങൾ (Unsuitable Actions)**
What actions should be avoided

### 7. **പ്രായോഗിക പ്രയോഗം (Practical Application)**
Real-world examples of how to apply this rule

### 8. **മുന്നറിയിപ്പുകൾ (Warnings)**
Important cautions and warnings

## 📖 EXAMPLE: SUNDAY (ഞായറാഴ്ച)

**SOURCE:**
> "Haftanın her gün ve gecesini, yedi klasik gök cisminden birisi yönetir."
> 
> Book: Havâss'ın Derinlikleri — I. Kitap  
> Page: 50  
> Chapter: GÜNDÜZ VE GECE SAATLERİ

**മലയാളം വിശദീകരണം:**

**പേര്:** സൂര്യൻ - ഞായറാഴ്ച

**അർത്ഥം:** ആഴ്ചയിലെ ഓരോ ദിവസത്തെയും രാത്രിയെയും ഏഴ് ഗ്രഹങ്ങളിൽ ഒന്ന് ഭരിക്കുന്നു. ഞായറാഴ്ചയുടെ അധിപൻ സൂര്യനാണ്.

**ഉപയോഗം:** ഞായറാഴ്ച സൂര്യന്റെ മണിക്കൂറുകളിൽ താഴെ പറയുന്ന കാര്യങ്ങൾ ചെയ്യാൻ അനുയോജ്യമാണ്.

**ഗുണങ്ങൾ:**
- ധനകാര്യങ്ങൾ മെച്ചപ്പെടാൻ
- പ്രതീക്ഷകൾ സഫലമാകാൻ
- ഉന്നത ഉദ്യോഗസ്ഥരിൽ നിന്ന് അംഗീകാരം ലഭിക്കാൻ
- സൗഹൃദങ്ങൾ വളരാൻ
- ശത്രുക്കളെ നേരിടാൻ
- ശത്രുവിനെ മിത്രമാക്കാൻ
- കായിക വിജയങ്ങൾക്ക്
- ശാരീരിക ശക്തിക്ക്
- എല്ലാവരുടെയും പ്രശംസ പിടിച്ചുപറ്റാൻ

**ചെയ്യേണ്ട കാര്യങ്ങൾ:** ധനം, പ്രതീക്ഷ, ഭരണാധികാരികളിൽ നിന്ന് അംഗീകാരം, സൗഹൃദം, ശത്രുക്കളെ നേരിടൽ, കായിക വിജയങ്ങൾ

**ഒഴിവാക്കേണ്ട കാര്യങ്ങൾ:** സൂര്യന്റെ സ്വഭാവത്തിന് യോജിക്കാത്ത കാര്യങ്ങൾ (ഉദാ: രഹസ്യ പ്രവർത്തനങ്ങൾ)

**പ്രായോഗിക പ്രയോഗം:** ഞായറാഴ്ച രാവിലെ സൂര്യോദയ സമയത്ത് ആരംഭിക്കുന്ന ആദ്യ മണിക്കൂർ സൂര്യന്റെ മണിക്കൂറാണ്. ഈ സമയത്ത് പുതിയ സംരംഭങ്ങൾ ആരംഭിക്കാം, ധനകാര്യ ഇടപാടുകൾ നടത്താം, ഉന്നത ഉദ്യോഗസ്ഥരെ കാണാം.

**മുന്നറിയിപ്പുകൾ:** സൂര്യന്റെ സ്വഭാവം തെളിച്ചവും ശക്തിയുമാണ്. അതിനാൽ രഹസ്യമായി ചെയ്യേണ്ട കാര്യങ്ങൾ ഈ ദിവസം ഒഴിവാക്കുക.

## 🔄 FUTURE PDF IMPORTS

**Rule:** Every future PDF imported for Astro Clock will automatically generate Malayalam explanations below each original source text.

**Process:**
1. Extract rule from PDF (preserve original text)
2. Store source metadata (book, page, chapter)
3. Generate detailed Malayalam explanation
4. Store both together in knowledge base
5. Display both in UI

**Principle:** Original text is NEVER replaced. Malayalam is added as explanation only.
